• The format of the file must be .xlsx.
  • Following columns must be included in the file: Name, Email, Phone, PIN, Role and Store.
  • Email addresses and phone numbers have to be unique, as these will be used to send invitation links to the users.
  • PIN is the only optional column. It is used to login the mobile application. If PIN is entered, it will be sent via text message. If it is not entered for a user, then a random 4-digits PIN will be sent.
  • The added user must exist in at least one {{nameConfig?.region | lowercase}}. In the Store column, specify the {{nameConfig?.region_plural | lowercase}} in which the user works in, separating them with a semicolon(;).
